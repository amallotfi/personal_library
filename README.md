# Simple Library Manager 📚

A beginner-friendly Python script to manage a personal book collection, wishlist, and donations.

## 💡 The Idea
The goal of this project is to simulate a real-life library management flow where a user can:
1. Add new books they own.
2. Maintain a wishlist of books they want to read.
3. Move a book from the **Wishlist** to the **Library** once it's acquired.
4. **Donate** books by removing them from the library and tracking them in a separate donation list.

## 🛠️ What I Used & Learned
In this project, I practiced core Python concepts:
- **Lists**: Creating, appending (`.append()`), and removing (`.remove()`) items.
- **User Input**: Using `input()` to interact with the user.
- **Conditionals**: Using `if` statements to handle logic and skip empty actions.
- **F-Strings**: Modern string formatting to display dynamic data.
- **Logic Flow**: Managing data movement between different lists without using loops.

## 🚀 How to Run
Simply run the script using Python:
```bash
python library.py
```
